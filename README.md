# R&S Studio

**Waves · Worlds · Wonder**

The official website for **R&S Studio** — a collaborative music production house specializing in immersive game soundtracks, animation scores, and high-energy arrangements.

🔗 **Live Site:** [r-s-studio.github.io](https://r-s-studio.github.io/)

## About

R&S Studio is a two-person production team:

- **Safara (Jiamu Shangguan)** — Composer. 10+ years of improvisational piano and arrangement experience, providing the melodic and emotional core for every project.
- **Rouri (Ray Zhang)** — Producer. 4 years of professional arrangement, mixing, and mastering experience, specializing in J-Pop, Hyperpop, and aesthetic electronic textures. 6,000+ followers.

We transform raw inspirations into industry-standard auditory experiences through a full-scale production pipeline.

## Features

- Responsive single-page design with smooth scroll navigation
- Interactive style-based project browser (Glitch / Piano / Acoustic / Atmosphere / Emo / Game OST)
- Embedded audio player for track previews
- Social links and contact section for both producers

## Project Categories

| Category | Description |
|---|---|
| **Glitch / Hyper** | Cyber, distorted, high-energy beats (breakcore, glitchcore, hyperpop) |
| **Piano Core** | Melodic center and harmonic depth (lo-fi, guofeng, emotional piano) |
| **Acoustic Folk** | Wood-guitar intimacy and warmth |
| **Atmosphere / Soundscape** | Scene-first ambient mood design |
| **Emo / Anime Vibes** | Youthful emotional narrative (emoplugg, anime type beats) |
| **Game OST** | Scores built for interactive worlds |

## Tech Stack

- HTML5 / CSS3 / JavaScript

## Local Development

Clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/R-S-Studio/r-s-studio.github.io.git
cd r-s-studio.github.io
open index.html
```

No build step required — it's a static site.
